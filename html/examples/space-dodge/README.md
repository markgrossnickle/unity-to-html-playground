# Space Dodge

Vertical-scrolling space dodger. The ship sits in the lower third of the
screen; touch or drag anywhere to steer it horizontally. Asteroids spawn at
the top and drift downward. Survive as long as you can — three lives, score
ticks up for every asteroid you clear.

## Run

From `html/`:

```sh
npm run dev
```

Then open the printed URL and pick "Space Dodge".

## Mechanics

- The ship's **y is fixed** at `height − 110`. The player only controls **x**,
  via lerp toward the most recent pointer x (~12% per frame).
- Asteroids spawn every 600 ms initially, ramping down to 200 ms as the score
  climbs.
- Scroll speed starts at 200 px/s and ramps to 400 px/s.
- Collision is a circle-circle test against the ship's ~24 px hitbox.
- A hit costs a life, shakes the camera, clears asteroids within 220 px of the
  ship, and grants ~1.5 s of invincibility (with a blink).
- Score increments by 1 per asteroid that clears below the ship.
- Three lives lost → game-over panel with a Restart button.
