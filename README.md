# unity-to-html-playground

Sandbox for two parallel experiments:

- `unity/` — research on running/porting Unity 2D animations to HTML/web (Unity WebGL, third-party exporters, sprite-sheet pipelines).
- `html/` — a Phaser-based coloring book game (tap-to-fill within line art) with a color palette.

Each subfolder has its own research / plan markdown.
